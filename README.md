# Modern Creative Portfolio Website

A highly aesthetic, responsive, and modern personal portfolio website built with pure HTML5, CSS3, and JavaScript.

## Features

- **Sleek Dark Mode Aesthetics**: Using a curated color palette (violet and cyan accent glows) with modern glassmorphism UI card styling.
- **Dynamic Typing Effect**: Cycles through skill statements in the Hero section.
- **Smooth Scroll & Navigation**: Float-scrolling headers and active navigation link indicators based on viewport position.
- **Interactive Components**: Floating overlays, custom scrollbar styling, active state animation, and dynamic stats card.
- **Fully Responsive**: Carefully tuned media queries for seamless display across mobile, tablet, and desktop screens.
- **Simulated Contact Form**: Elegant, interactive forms utilizing active indicators and visual status response toasts.

## Technology Stack

- **HTML5**: Semantic tags, accessibility (ARIA attributes), SEO meta tags.
- **CSS3 Custom Properties**: Dynamic variables, layout systems (Flexbox, Grid), transition effects.
- **Vanilla JavaScript**: DOM Manipulation, event handling, responsive menu toggles.
- **Google Fonts & FontAwesome**: Integration of modern typography and icon packs.

## Getting Started

1. Clone this repository or download the zip:
   ```bash
   git clone <your-repository-url>
   ```
2. Open `index.html` in your web browser, or use a local development server (e.g., Live Server in VS Code) to run the website.
